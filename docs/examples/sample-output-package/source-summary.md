## 来源

- 来源类型：方法论文章 / pasted-text

## 核心观点

这篇文章不是讲“多问几轮 AI”，而是讲如何把临时对话沉淀成一个可复用工作区：有项目目录、规则、skills、台账、复盘和发布包。
