# 星禾示例图集

这个文件集中展示 `assets/examples/` 下的示例图。README 首页只展示精选图片，这里用于完整浏览不同视觉形态的构图方向。

这些图片用于参考构图、信息层级、人物参与方式和平台比例，不要求每次复刻同一种画风。
