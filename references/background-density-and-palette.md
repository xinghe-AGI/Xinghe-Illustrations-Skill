# 背景信息密度与配色规则

本文件用于避免解释图、流程图、知识卡片和正文配图出现“大块空白、元素漂浮、颜色单调”的问题。它只学习参考图的布局、背景承载方式和颜色组织，不复制参考图人物、文字、水印、品牌、具体图标或完整构图。

## 核心原则

背景不是装饰，而是信息承载层。背景里的便签、夹子、胶带、桌面、笔记本、边栏提示、色块、分区线和小图标，都必须帮助读者理解主路径、阅读顺序、关键词、检查点或情绪状态。

不要把“清爽”理解成空白。清爽是主次清楚、阅读路径明确、辅助元素有秩序；不是白底上漂着几个框和人物。

## 背景密度等级

每个候选方向和 prompt 都要写 `background_density_level`。

| 等级 | 适用场景 | 规则 |
|---|---|---|
| `light` | 平台封面、单概念图、情绪主视觉 | 只保留 10%-15% 辅助背景元素，避免抢标题和人物 |
| `medium` | 正文配图、解释图、普通知识卡 | 20%-35% 区域有承载面：浅色分区、便签、边栏、桌面小物、短批注 |
| `rich` | 白板写作布局、流程解释、关系型知识卡、技术流程图 | 35%-50% 区域有辅助信息层，但主路径仍清楚 |
| `dense` | 全景信息图、信息图海报、系统地图 | 50% 以上区域可有分区和辅助信息，但必须有明确阅读路径和层级 |

默认选择：

- 公众号封面、小红书封面：`light` 或 `medium`。
- 情绪锚点图：`medium`，让背景表达压力、混乱、释然等情绪来源。
- 解释图、流程图、技术架构图、知识卡片：`medium` 或 `rich`。
- 全景信息图、信息图海报：`rich` 或 `dense`。

## 可用背景信息层

优先使用低科技、手写、桌面和白板类元素：

- 白板边框、白板槽、板擦、马克笔、磁吸夹、夹子、胶带、图钉。
- 桌面、笔记本、便签、便利贴、索引卡、纸条、流程卡、检查清单。
- 侧边栏提醒、底部工具带、角落小结、短句提示、手写问号卡。
- 浅色分区底、虚线边界、点阵底、细分隔线、编号圆点、短箭头。
- 与主题相关的轻量小图标：灯泡、目标、齿轮、数据库、文件、放大镜、回流线、检查框。

背景元素要少量多点分布，而不是堆成一团。每张图可选 3-6 类辅助元素；全景信息图可以更多，但要分区。

## 白板写作布局

当内容是流程、解释、方法、决策或推理时，优先考虑“白板 + 桌面 + 便签”的写作布局：

- 上方放主标题或核心问题，保持 1-2 行。
- 中部放主路径、节点、卡片或对比区。
- 边缘放 2-4 个短提示便签，承接注意事项、前提、检查点或结论。
- 底部可放桌面层：笔、胶带、便签、卡片、打开的笔记本，用于减少空白和增强创作现场感。
- 星禾可以在左下、右侧或角落执行动作：贴卡、拉线、圈重点、递标签、整理节点；结构图里人物可缩小或只露手。

## 配色规则

冷白或极浅蓝灰仍是底色，但不要只使用蓝、橙、黑三种颜色。允许加入低饱和辅助色，让画面更有层次：

- 深蓝黑 / 黑色：主线稿、主标题、关键节点文字。
- 蓝色 / 蓝灰：背景分区、反馈线、AI 状态、二级说明。
- 橙色：标题中的关键词、主路径、短标签文字、行动线。
- 鼠尾草绿：步骤编号、稳定状态、方法卡、通过项。
- 柔和黄色：提醒便签、灯泡、重点提示、临时想法。
- 淡紫 / 紫蓝：分支、对比项、策略层、可选路径。
- 珊瑚粉 / 低饱和红：风险、卡点、冲突、情绪压力。
- 暖纸色只可作为小面积便签或卡片底，不做大面积背景。

颜色要服务文字层级和阅读路径。可以让文字本身使用橙色、绿色、紫色或蓝色，但同一张图里不应超过 4-6 个主要文字颜色；主正文和关键标签仍要可读。

## 留白规则

- 允许留白，但留白要有用途：给标题呼吸、给主路径停顿、给人物动作留空间。
- 解释图、流程图、知识卡片和全景图不允许出现大块无功能空白。
- 如果画面右侧、上方或底部明显空，应补充边栏提示、短便签、工具带、浅分区或桌面元素。
- 如果背景元素让文字变小、路径变乱或标题不突出，优先删背景，不缩小主信息。

## Prompt 必填字段

每个候选方向和 prompt 至少写：

- `background_density_level`：`light` / `medium` / `rich` / `dense`
- `supporting_background_elements`：列出 3-6 个辅助背景元素
- `palette_plan`：主文字色、辅助色、路径色、风险色和背景分区色
- `blank_space_control`：说明哪些区域保留留白，哪些区域用背景信息层承接

## 禁忌

- 不复制参考图中的人物、文字、水印、品牌、具体图标和完整排版。
- 不把背景做成商业海报、课程 PPT、真实 UI、复杂仪表盘或儿童贴纸墙。
- 不使用大面积黄色、米色、暖灰、纸纹、阴影、渐变或复古纸张质感。
- 不为了“丰富”堆满随机小物件；背景元素必须和主题、阅读路径或情绪有关。
- 不把所有文字统一成橙色，也不把橙色只当下划线。橙色可以是文字颜色，但要有节制。
