# 封面构图模板

本文件只在用户要求公众号封面、小红书封面或平台首图时读取。模板用于稳定输出，不要复刻某一张旧图。

## 公众号封面构图

### 左标题右行动

- 左侧 40%-50% 放短标题。
- 右侧放星禾和一个主题物件：文件盒、方法卡、流程线、小工作台。
- 适合：观点文、方法论、项目复盘。
- 星禾动作：整理、托举、递出、圈出、把散落内容放回盒子。
- 参考图：`assets/examples/15-wechat-left-title-right-action.png`。

### 中标题下轻场景

- 标题在中左或中上，下面是一条轻量动作线。
- 星禾在右下或中右处理这条线。
- 适合：系统、流程、自动化、工作区主题。
- 注意：不要变成正式流程图。
- 参考图：`assets/examples/17-wechat-process-line.png`。

### 宽留白单物件

- 大面积白底，标题一侧，星禾与单个物件另一侧。
- 适合：抽象判断、品牌感文章、系列封面。
- 物件只保留 1 个核心隐喻，例如文件夹、桥、台账、灯箱、卡片盒。
- 参考图：`assets/examples/16-wechat-wide-white-space.png`。

## 小红书封面构图

### 上标题下人物

- 上方 45%-55% 放 2-4 行大标题。
- 下方放星禾和动作场景。
- 适合：经验帖、方法帖、复盘帖。
- 重点：标题像打字排版，星禾动作承接主题。
- 参考图：`assets/examples/18-xhs-typed-title-bottom-xinghe.png`。

### 标题卡片 + 星禾小场景

- 白底或浅点阵底，标题像卡片里的大字标题。
- 关键词可加一条粉紫或橙色下划线。
- 底部星禾整理盒子、托卡片、拉流程线。
- 适合：小红书首图和系列封面。
- 参考图：`assets/examples/19-xhs-keyword-underline-card.png` 或 `assets/examples/20-xhs-title-card-method-stack.png`。

### 左上标题右下承托

- 标题从左上开始，星禾在右下用动作托住主题物件。
- 适合：标题较短、需要更强人物记忆点的封面。
- 注意：右下不能太重，保留底部安全边距。

### 中心大标题 + 底部微动作

- 标题居中偏上，人物和物件在底部作为视觉签名。
- 适合：强判断、强结果标题。
- 注意：人物不能太小，否则星禾 IP 记忆点不足。

## 构图安全区

- 小红书：标题和人物离四边至少留 6%-8% 画布边距；底部按钮或系统裁切区不要放关键文字。
- 公众号：标题和人物离左右边至少留 5% 画布边距；移动端分享缩略图里标题仍要可读。
- 人脸、手、核心物件不要被标题遮挡。
- 不把画面主体压到最底部或最右边。

## 反复刻规则

- 可以参考用户提供截图的“排版关系”，不能照搬背景、人物、文字样式和具体插画。
- 可以使用 `assets/examples/00-xinghe-ip-baseline.png` 锚定星禾形象；场景参考图只用于动作密度、留白和线条。
- 每次都要回答：这张封面里星禾正在亲手承托哪个标题判断？
